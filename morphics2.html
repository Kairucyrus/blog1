INDUSTRIAL ATTACHMENT.

Intro

Industrial attachment aims at equipping students with practical experience and skills required in a certain field. Students' exposure to demands and challenges of a work place is necessary and not to be left out.

Skills and expectations

There came C language, there came the object-oriented languages as C++ and java. That was not all. Worse or better yet, I had to handle Java and Python altogether. Thereafter, web development found it way through. Afterwards Python took over with a little light-shining on machine learning and data analysis. The acquisition of skills has been commendable. Though the beginning seemed small and easily despisable, the progress has been good, fairly good.
